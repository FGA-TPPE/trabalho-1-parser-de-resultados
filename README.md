## Uso

Para baixar as dependências:

```
$ mvn dependency:resolve
```

Para compilar:

```
$ mvn compile
```

Para executar:

```
$ mvn exec:java -Dexec.mainClass="parserapp.Main"
```

Para rodar os testes:

```
$ mvn test
```

Os arquivos de saída estarão localizados em: "./src/files"
